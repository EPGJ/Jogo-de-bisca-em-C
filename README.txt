Trabalho de Estrutura de Dados 1 - Jogo de Bisca
	
Jogo de Bisca desenvolvino na linguagem de programação C. O jogo não possui interface gráfica logo deverá ser executado no terminal do Linux.

--> Comandos necessários para execução do jogo:

-- Abra o terminal : Ctrl + Alt + T
-- Navegue até a pasta do jogo: cd <caminho_para_o_diretório>. 
  (Ou entre na pasta do jogo e pressione o botão direito do mouse e clique em abrir no terminal)
-- Compilar o jogo: make
-- Abir executável: ./trabalhoED1
